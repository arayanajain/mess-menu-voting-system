# Mess Menu Voting System

A command-line voting system built in C that lets users vote for their favourite mess menu dishes.

## Features
- View mess menu
- Vote for dishes
- Display vote counts
- View voting history
- Rank top dishes
- Reset votes

## How to Run
```bash
gcc -o voting_app main.c voting.c
./voting_app
```

## Tech Stack
- Language: C
- Concepts: Structs, Arrays, Sorting (Bubble Sort), File I/O
